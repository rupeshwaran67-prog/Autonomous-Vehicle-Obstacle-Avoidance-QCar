# Autonomous-Vehicle-Obstacle-Avoidance-QCar
Autonomous Vehicle Obstacle Avoidance using QCar Digital Twin
Project Overview

This project implements an autonomous lane-following and obstacle-avoidance system using the Quanser QCar Digital Twin within the Quanser Interactive Labs (QLabs) simulation environment.

The system was developed using MATLAB/Simulink and Python to demonstrate perception, control, and decision-making capabilities for autonomous vehicles operating in a highway environment.

Objectives
Implement autonomous lane following
Detect obstacles using onboard sensors
Execute safe obstacle avoidance manoeuvres
Return to the original lane after obstacle clearance
Maintain stable vehicle behaviour at nominal speed

Simulation Scenario

The vehicle operates within the Open Road Highway environment.

The implemented algorithm performs:

Lane following
Obstacle detection
Lane departure for avoidance
Safe obstacle bypass
Return to original lane
Vehicle stop after task completion
